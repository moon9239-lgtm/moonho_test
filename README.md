# moonho_test GitHub Practice

이 저장소는 Git과 GitHub를 말로 시키면서 익히는 연습장입니다.

## 가장 자주 쓰는 말

| 내가 말할 말 | 실제로 하는 일 |
| --- | --- |
| 상태 확인해줘 | 바뀐 파일이 있는지 확인 |
| 뭐가 바뀌었는지 보여줘 | 변경 내용을 비교 |
| 저장해줘 | 현재 변경사항을 commit |
| 올려줘 | commit을 GitHub에 push |
| 기록 보여줘 | 저장 기록 확인 |
| 최신 내용 가져와줘 | GitHub의 최신 내용을 pull |

## 오늘의 실습 순서

1. `practice/shopping-list.txt`에 사고 싶은 물건 하나를 추가합니다.
2. "상태 확인해줘"라고 말합니다.
3. "뭐가 바뀌었는지 보여줘"라고 말합니다.
4. "저장해줘"라고 말합니다.
5. "올려줘"라고 말합니다.
6. GitHub 웹사이트에서 파일이 바뀌었는지 확인합니다.

## 브랜치 실습

브랜치는 새로운 작업 공간입니다. 나중에 이렇게 연습합니다.

1. "새 가지 만들어줘. 이름은 practice-change로 해줘"
2. 파일을 수정합니다.
3. "저장해줘"
4. "올려줘"
5. "PR 만들어줘"

## 추천 연습 문장

아래 문장만 익히면 GitHub 작업의 대부분을 말로 처리할 수 있습니다.

```text
상태 확인해줘
변경 내용 보여줘
저장해줘
올려줘
기록 보여줘
```
