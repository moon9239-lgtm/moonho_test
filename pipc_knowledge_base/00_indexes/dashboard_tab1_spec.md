# 1번 탭: 전체회의 통계분석 대시보드 명세

## 목표

개인정보보호위원회 전체회의의 회의, 안건, 결정문, 처분, 조항, 발언, 위원 활동을 한눈에 볼 수 있는 첫 화면이다. 숫자는 직원이 빠르게 흐름을 파악하고, 클릭하면 안건 통합검색과 회의 상세 화면으로 이동하는 구조를 전제로 한다.

## 클릭 이동 설계

| 클릭 대상 | 이동 |
| --- | --- |
| 연도 | 2번 탭 안건 통합검색, `year` 필터 |
| 의결/보고 | 2번 탭 안건 통합검색, `agenda_kind` 필터 |
| 공개/비공개 | 2번 탭 안건 통합검색, `visibility` 필터 |
| 처분 유형 | 2번 탭 결정사건 검색, `sanction_kind` 필터 |
| 조항 | 2번 탭 조항별 검색, `article_raw` 필터 |
| 주제 | 2번 탭 유사사건/주제 검색 |
| 위원 | 4번 탭 위원 대시보드 |
| 특정 회의/안건 | 3번 탭 회의 상세 |
