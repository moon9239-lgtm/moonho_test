# Image prompt template

## 캐릭터 시트 프롬프트

```text
Original fictionalized 2D animated character for a Korean public policy committee meeting, not a portrait, no photographic likeness, no real-person face matching.
Use the provided reference photo only for broad visual cues such as silhouette, hairstyle, glasses, and professional clothing tone. Create a stylized original character, not a photorealistic or exact likeness.

Character source:
- Internal name: {name}
- Committee role: {role}
- Meeting function: {meeting_function}
- Expertise motif: {core_motif}
- Reference photo: {reference_photo}
- Visual anchor: {visual_anchor}
- Main prop: {prop}
- Palette: {palette}

Create a clean editorial animation character sheet:
- front view, three-quarter view, seated meeting pose, speaking pose
- expression set: neutral listening, careful question, mild disagreement, summarizing, concern, agreement
- professional public-sector clothing, modern Korean conference room context
- simple readable silhouette, restrained detail, consistent proportions
- no logos, no real institution seal, no text labels in the image
- keep the character original and symbolic, based on role and expertise rather than real appearance

Style:
clean 2D animation, polished line art, soft but not cute, civic technology documentary tone, balanced color accents, white or light gray background
```
