# PIPC member character bible

이 문서는 모의회의 애니메이션에서 쓰기 위한 캐릭터 설정 초안이다. 원본 멤버 프로필과 발언 분석을 바탕으로 했지만, 실제 외모나 목소리를 재현하지 않는 오리지널 아바타로 설계한다.

## 공통 아트 디렉션

- 스타일: 공공정책 다큐멘터리풍 2D 모션그래픽, 선명한 실루엣, 과장되지 않은 표정
- 기본 의상: 현대적인 회의복, 명찰이나 기관 로고는 쓰지 않음
- 카메라: 흉상 컷, 회의 테이블 중간 샷, 발언자 클로즈업
- 공통 표정 6종: 경청, 확인 질문, 조심스러운 반대, 정리, 우려, 동의
- 공통 제스처 4종: 자료 짚기, 펜으로 밑줄 긋기, 손바닥을 열어 조율하기, 고개 끄덕이며 메모하기
- 금지: 실존 인물 사진 닮게 만들기, 실제 목소리 모사, 풍자적 왜곡, 로고 사용
